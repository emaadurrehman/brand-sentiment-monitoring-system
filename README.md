# Brand Sentiment Monitoring System

An end-to-end automated sentiment analysis pipeline for food delivery platforms using web scraping, NLP, workflow automation, and Docker containerization. Analyzes 900+ news articles across DoorDash, Uber Eats, Grubhub, and Foodpanda to provide multi-dimensional brand reputation insights.

**Key Features:** Selenium web scraping | GNews API integration | VADER/TextBlob sentiment analysis | Aspect-based sentiment | Time-series analysis | Word cloud generation | n8n workflow automation | Docker deployment
